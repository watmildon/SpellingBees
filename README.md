# Spelling Bees

A children's spelling game. An emoji is shown and the player types the matching word to free the bees.

## How to Play

1. Look at the emoji and type the word it represents.
2. Correct letters are revealed left-to-right, freeing a bee for each one.
3. Click the speaker button to hear the word pronounced.
4. Spell the full word to free all the bees and advance to the next round.
